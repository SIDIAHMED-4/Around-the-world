# Around the World üåç

**Around the World** is a modern React.js application that lets users explore countries around the world.  
The app supports **English** and **Arabic** using the `i18next` library for internationalization and uses **Tailwind CSS** for a responsive and stylish UI.

## Features
- Explore information about countries worldwide.
- Fully bilingual support: English & Arabic using `i18next`.
- Responsive and modern design with **Tailwind CSS**.
- Smooth navigation and animations.
- Easy to extend with more countries or additional languages.

---

## Technologies
- **Frontend:** React.js  
- **Styling:** Tailwind CSS  
- **Internationalization:** `i18next` (i18n)  
- **Hosting/Deployment:** Vercel

## Installation

1. Clone the repository:
bash
git clone https://github.com/SIDIAHMED-4/around-the-world.git

2. Navigate to the project folder:

cd around-the-world

3. Install dependencies:

npm install

---
## Usage

Start the development server:

npm start

Open http://localhost:3000
 in your browser.

Switch between English and Arabic using the language selector.
